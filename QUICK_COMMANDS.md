## 💡 使用技巧

### 1. 链式调用
可以用 `→` 连接多个指令：
```
analyze code → review 安全性 → 给出改进建议
```

### 2. 条件指令
使用 "如果...则..." 结构：
```
review 代码，如果有安全问题则详细说明
```

### 3. 范围限定
明确指定范围：
```
review scripts/ 目录下的所有脚本
```

### 4. 输出控制
指定输出格式：
```
分析结果用表格形式呈现
```

### 5. 上下文引用
引用之前的讨论：
```
基于上述 review 结果实现改进
```
