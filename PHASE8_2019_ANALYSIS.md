## 💡 失效机制分析

### ❌ 否定："结构分化市失效"假设

**不符合条件**：
1. ❌ CV=0.61（中等分化，非高分化）
2. ❌ 7只股票全部正收益（无崩盘股）
3. ❌ 没有明显的行业轮动（消费、金融、医药、公用事业均上涨）

### ✅ 新假设："牛市初期动量滞后"

**机制分析**：

**问题1：初期遗漏真牛股**
- 恒瑞1月未选中（技术面滞后于基本面）
- 基本面驱动的牛股，技术信号出现较晚

**问题2：调整期错杀强势股**
- 4月市场调整，MA5跌破MA10
- **动量策略机械止损**，清仓五粮液、恒瑞
- 但这只是**健康回调**，非趋势反转

**问题3：反向选股**
- 5月选中长江电力（公用事业，防御性）
- **错误判断**：以为市场转弱，选择防御性股票
- 实际：牛市继续，长江电力涨幅垫底

**问题4：频繁交易摩擦**
- Q3-Q4反复买卖
- 错过持续上涨（Buy and Hold 策略更优）

### 为什么固定持仓跑赢？

**固定持仓优势**：
- 持有全部7只股票，不做调整
- **自动持有强势股**（五粮液+151%、恒瑞+97%、茅台+92%）
- **避免动量滞后**（不会在调整期卖出）
- **避免频繁交易**（零换手）

**结果**：
- 固定持仓 = 7只股票平均收益 = +74.31%
- 动态选股 = 多次清仓强势股 + 换入弱势股 = +20.80%

## 🔬 对假设的影响

### 原假设调整

**Phase 8原假设**（2021分析得出）：
```
策略在"结构分化市"失效
- 风格一致市：策略跑赢
- 结构分化市：策略跑输
```

**2019年验证结果**：
```
❌ 部分否定：
- CV=0.61（中等分化，非高分化）
- 但策略跑输-17.15%

✅ 新发现：
- 失效原因是"牛市初期动量滞后"，非"分化"
- 短期技术信号在基本面驱动的牛市中失效
```

### 修正后假设

**假设v2.0**：
```
策略失效场景：
1. 牛市初期（2019）：动量滞后于基本面，错杀强势股 ❌
2. 结构分化市（2021/2025）：动量失配行业轮动 ❌
3. 震荡/熊市（2022/2023）：动量有效，策略跑赢 ✅
```

**共性问题**：
- **动量策略的本质缺陷**：用过去20日预测未来，存在滞后
- **牛市初期**：基本面先行，技术面滞后 → 错过启动
- **分化市**：行业轮动快，技术信号来不及切换 → 失配
